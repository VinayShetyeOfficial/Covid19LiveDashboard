### Prerequisites

- Node.js 16.x or higher

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/VinayShetyeOfficial/Covid19-Live-Dashboard.git
   ```
2. Navigate to the project directory:


